Sonic-R

Sonic-R/Sonic Runner (you can choose how to call it), is a fan-project developted by PlopesK using HTML, CSS and JS, based on the Dino Game from Chrome.
This project was created with love for the Sonic franchise with no intended monetary values :D.
I hope I can still update the project with time, adding new playble characters, new stages, even new mechanics! So I hope everyone seeing this to have the best fun possible playing this "website game" :)

Development

The idea of the project is very simple, but it demanted a lot of work since I didn't really was customized with the JS logic (I already had a lot of knowledges about HTML and CSS), so it was really useful to motivate me to study more.
The problem really was that I had to create and adapt a new logic to the game, being really different to an 'copy/paste' project, Demanding to use only my logic to create it.
Searching for others dinosaurs games in the internet, for example, use mainly the 'canvas' element, with I choosed to not use (since when I discovered his existency, the project was already in a really long way, so probrably I would have to recode everything :,v ).

Even if it isn't the most well-coded project in existance, neighter the most impressive or something a-like. But, I'm certain that I learned a lot producing it :), and it's a love letter from a fan to others fans of the Sonic franchise . So, I hope others can find something useful in it or have fun playing it :D.

Sprites

There's only an section focused on sprites because I wanted to clarify how annoying was to cut and implement those graffics :,). I had to searching by the internet and recort every-single-sprite manually to recreate the stages and actions made by the characters.
For that, I used the Piskel website to create every gif and even modify some sprites to work in the project.
Most of the sprites where cutted from Sonic the Hedgehog 1, but the Sonic model is a mix between every sprite of Sonic from the Sonic Advance Series.
Most of the recorted spritesheets were downloaded from the Spriters Resource website , so a lot of thanks to everyone who uploaded those sprites, since that helped a lot :D.

Curiositys

Both Sonic-R and Sonic Runners are already existing Sonic games!
Sonic-R was a racing game developed for Sega Saturn and ported to many collections of games from the franchise, for example the Sonic Gems Collection for Playstation 2. As a kid I played it a lot and loved everything about it, even if the critics didn't liked it that much :,v.

Sonic Runners was an free mobile game that ended service for the creation of Sonic Runners Adventure, a paid mobile game with didn't made that much success.
It's really sad that it ended that recently after lauching :,v. It was a really fun experience, would be glad if it returned someday. But, even if it didn't returned officially, Sonic Runners received an fan-project called Sonic Runners Revival, wich is an amazing work made by fans to recreate Sonic-Runners :D.



The project itself demanded a lot of hard work to be created (but still, it was really fun to develope), so I really hope you had atleast a bit of fun playing it :D
